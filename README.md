# QuickMart - E-commerce Application

A fully functional e-commerce web application built with Laravel for backend internship application.

## Features
- User Authentication (Login/Register)
- Product Catalog with Categories
- Shopping Cart System
- Responsive Design with Bootstrap
- MySQL Database with Seed Data

## Tech Stack
- **Backend:** Laravel 12, PHP 8.3
- **Frontend:** Bootstrap 5, Blade Templates
- **Database:** MySQL 8.4
- **Development:** Laragon

## Setup Instructions
1. Clone repository: `git clone https://github.com/username/quickmart.git`
2. Install dependencies: `composer install && npm install`
3. Configure `.env` file with database credentials
4. Run migrations: `php artisan migrate --seed`
5. Start server: `php artisan serve`

## Purpose
Built for Backend (Laravel) Internship application at Technorio Inc.
Demonstrates Laravel MVC, Database Design, Authentication, and E-commerce functionality.
